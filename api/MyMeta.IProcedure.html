Interface IProcedure

IProcedure represents a view or function in your DBMS.

Namespace: MyMeta
Assembly: OpenMyMeta.dll
Syntax
[Guid("57DD1E11-A401-4EC4-B2D6-EB766D0C9B97")]

public interface IProcedure
Remarks

IProcedure Collections:

ParametersContains all of the parameters for the procedure
ResultColumnsContains an IResultColumn for each column return by the procedure
PropertiesA collection that can hold key/value pairs of your choosing
GlobalPropertiesA collection that can hold key/value pairs of your choosing for all Procedures with the same Database
AllPropertiesA collection that combines the Properties and GlobalProperties Collections
Examples

VBScript

output.writeLn "This Procedure takes   " + objProcedure.Parameters.Count    + " Parameters"
output.writeLn "This Procedure returns " + objProcedure.ResultColumns.Count + " ResultColumns"

Alias

You can override the physical name of the Procedure. If you do not provide an Alias the value of 'Procedure.Name' is returned. If your Procedure in your DBMS is 'Q99AAB' you might want to give it an Alias of 'Employees' so that your business object names will make sense. You can provide an Alias the User Meta Data window. You can also set this during a script and then call MyMeta.SaveUserMetaData(). See Name

AllProperties

AllProperties is essentially a read-only collection consisting of a combination of both the Properties (local) collection and the GlobalProperties (global) collection. The local properties are added first, and then the global properties are added however, only global properties for which no local property exists -- are added. This makes local properties overlay global properties. Global properties can act as a default value which can be overridden by a local property. See IProperty.

GlobalProperties

The Properties for all Procedures within the same Database. These are user defined and are typically stored in 'UserMetaData.xml' unless changed in the Default Settings dialog. Properties consist of key/value pairs. You can populate this collection during your script or via the Dockable window. To save any data added to this collection call MyMeta.SaveUserMetaData().
