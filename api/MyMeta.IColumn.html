Interface IColumn

IColumn represents a column or field in your DBMS. IColumn is used to represent columns for both Tables and Views.

Inherited Members

public interface IColumn : IEquatable<IColumn>
Remarks

IColumn Collections:

ForeignKeysContains all of the foreign keys this column plays a role in
PropertiesA collection that can hold key/value pairs of your choosing
GlobalPropertiesA collection that can hold key/value pairs of your choosing for all Columns with the same Database
AllPropertiesA collection that combines the Properties and GlobalProperties Collections
Examples

VBScript

Dim objColumn
Set objColumn = objTable.Columns.Item(5)
Set objColumn = objTable.Columns.Item("FirstName")

' Loop through the collection
For Each objColumn in objTable.Columns
    output.writeLn objColumn.Name
    output.writeLn objColumn.Alias
    output.writeLn objColumn.DataTypeNam
Next

JScript

var objColumn;
objColumn = objTable.Columns.Item(5);
objColumn = objTable.Columns.Item("FirstName");

for (var j = 0; j < objTable.Columns.Count; j++) 
{
    objColumn = objTable.Columns.Item(j);

    output.writeln(objColumn.Name);
    output.writeln(objColumn.Alias);
    output.writeln(objColumn.DataTypeName);				
}

Alias

You can override the physical name of the Column. If you do not provide an Alias the value of 'Column.Name' is returned. If your column in your DBMS is 'TXT_FIRST_NAME' you might want to give it an Alias of 'FirstName' so that your business object property will be a nice name. You can provide an Alias the User Meta Data window. You can also set this during a script and then call MyMeta.SaveUserMetaData(). See Name

AllProperties

AllProperties is essentially a read-only collection consisting of a combination of both the Properties (local) collection and the GlobalProperties (global) collection. The local properties are added first, and then the global properties are added however, only global properties for which no local property exists -- are added. This makes local properties overlay global properties. Global properties can act as a default value which can be overridden by a local property. See IProperty.
