Interface IDatabase

IDatabase represents a databaase in your DBMS.

Namespace: MyMeta
Assembly: OpenMyMeta.dll
Syntax
[Guid("28390206-0D55-429D-8D17-C2DD1873AAE2")]
[InterfaceType(ComInterfaceType.InterfaceIsDual)]
public interface IDatabase
Remarks

IDatabase has 5 Collections:

TablesContains all of the tables for the databaase
ViewsContains all of the views the databaase
ProceduresContains all of the procedures for the databaase
DomainsContains all of the domains for the database
PropertiesA collection that can hold key/value pairs of your choosing
Examples

VBScript

Dim objDatabase
For Each objDatabase in MyMeta.Databases
    output.writeLn objDatabase.Name
    output.writeLn objDatabase.Alias
Next

JScript

var objDatabase;
for (var j = 0; j < MyMeta.Databases.Count; j++) 
{
    objDatabase = MyMeta.Databases.Item(j);

    output.writeln(objDatabase.Name);
    output.writeln(objDatabase.Alias);
}

Alias

You can override the physical name of the Column. If you do not provide an Alias the value of 'Column.Name' is returned. If your column in your DBMS is 'TXT_FIRST_NAME' you might want to give it an Alias of 'FirstName' so that your business object property will be a nice name. You can provide an Alias the User Meta Data window. You can also set this during a script and then call MyMeta.SaveUserMetaData(). See Name
